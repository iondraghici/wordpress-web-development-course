## Key Concepts

### WordPress
A powerful, free open-source content management system that powers millions of websites worldwide. Used for blogs, business sites, e-commerce, and more.

### Themes
Design templates that control the appearance of your WordPress website. Can be customized with CSS and custom code.

### Plugins
Extensions that add functionality to WordPress without modifying core files. Can be installed to add any feature you need.

### Posts vs Pages
- **Posts**: Chronologically ordered, typically for blog content with categories and tags
- **Pages**: Static content like About, Contact pages without chronological ordering
