## Key Concepts Summary

### WordPress
An open-source content management system (CMS) used to create and manage websites without coding.

### Dashboard
The admin interface where you manage all aspects of your WordPress website.

### Posts vs Pages
- **Posts**: Blog content, organized chronologically with categories and tags
- **Pages**: Static content like About, Contact, Services pages

### Plugins
Extensions that add functionality to your WordPress site.

### Themes
Design templates that control how your website looks.
